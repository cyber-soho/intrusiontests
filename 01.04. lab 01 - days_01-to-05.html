Disable IPv6 on your host machine's physical network adapter. IPv6 can bypass IPv4-only VPN tunnels and reveal your real IP address.

Verification & Threat Modeling

An untested security architecture is inherently insecure. Perform these validation checks to ensure zero leaks.

Verification Tests

  • 1. IP Leak Test: curl -s https://ipleak.net/json/
    Expected: Only Tor exit node IP. No real ISP data.
  • 2. DNS Leak Test: Visit dnsleaktest.com via Whonix.
    Expected: DNS servers belong to Tor/VPN infra only.
  • 3. WebRTC Test: Visit browserleaks.com/webrtc.
    Expected: Local IP is 10.152.x.x. No public IP.
  • 4. IPv6 Test: curl -6 https://ipv6.icanhazip.com
    Expected: Connection timeout / Network unreachable.

The Kill Switch Validation

Proves traffic stops if the VPN tunnel drops.

  1. Open Wireshark on host, capture WAN.
  2. Start continuous ping on Whonix WS: ping google.com
  3. In OPNsense, disable VPN1 instance.
  4. Observe: Pings must stop instantly. Wireshark must show NO unencrypted traffic on WAN.
